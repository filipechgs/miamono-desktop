# Checklist Final de Qualidade

## Build e empacotamento

- [ ] `npm run build` executado sem erros.
- [ ] `npm run package:dir` executado sem erros.
- [ ] Instalador Windows gerado.
- [ ] Artefato Linux (AppImage/tar.gz) gerado.

## Funcionalidade

- [ ] Cadastro de servicos funcionando.
- [ ] Cadastro de pagadores funcionando.
- [ ] Cadastro de recebimentos funcionando.
- [ ] Filtros dinamicos funcionando.
- [ ] Exportacao CSV funcionando.
- [ ] Exportacao PDF funcionando.

## Dados e regressao

- [ ] `npm run seed` executado.
- [ ] `npm run test:regression` executado.
- [ ] Validacao visual em dados 2024/2025 concluida.

## Documentacao e release

- [ ] `README.md` atualizado com download/instalacao.
- [ ] `CHANGELOG.md` atualizado.
- [ ] Versao no `package.json` revisada.
- [ ] Tag Git criada (`vX.Y.Z`).
- [ ] Release publicada com artefatos anexados.
